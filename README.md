## 🚀 Overview

This project analyzes the rapid growth of Electric Vehicle (EV) adoption globally using **Power BI**.  
It provides insights into **sales trends, regional performance, and powertrain distribution** to support data-driven decision-making.

---

## 🎯 Objectives

- Analyze global EV sales trends 📈  
- Identify top-performing countries 🌍  
- Compare EV types (BEV vs PHEV) ⚡  
- Track KPIs like total sales, growth rate, and market share  
- Understand key drivers of EV adoption  

---

## 🛠️ Tools & Technologies

- Power BI  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation

## 📊 Dashboard Features

### 🔹 Trend Analysis
- Year-wise EV growth visualization  
- KPI: Total Global Sales  

### 🔹 Regional Analysis
- Top countries contributing to EV adoption  
- Geographic comparison  

### 🔹 Powertrain Analysis
- BEV vs PHEV comparison

## 📈 Key Insights

- Global EV sales exceeded **17M in 2024**  
- China leads with ~**50% market share** 🇨🇳  
- BEVs dominate with ~**70% share** ⚡  
- Policy changes significantly impact EV adoption

- 
- Data inconsistency from multiple sources  
- Missing data in emerging markets  
- Complex DAX calculations
